# NSW Selective School Exam 学习平台 - 项目概述

## 项目背景

NSW（新南威尔士州）精英中学入学考试是澳大利亚重要的教育选拔考试，涵盖四个核心科目：
- **Reading**（阅读理解）
- **Writing**（写作）
- **Maths**（数学）
- **Thinking Skills**（思维技能）

## 项目目标

开发一个专门用于NSW精英中学考试准备的Web学习管理平台，帮助学生高效管理学习过程中遇到的难题和易错题。

### 核心价值主张

1. **便捷记录**：通过手机拍照快速上传题目，无需手动输入
2. **智能管理**：自动追踪难题、易错题，优先推荐复习
3. **多用户协作**：支持学生、家长、导师三种角色协同学习
4. **AI就绪**：为未来的AI辅助解题功能预留接口

## 目标用户

### 用户角色
- **Student（学生）**：主要使用者，进行题目练习和复习
- **Parent（家长）**：监督和支持学生学习进度
- **Tutor（导师）**：提供专业指导和题目讲解

## 核心功能

### 1. 题目管理系统
- **快捷上传**：手机拍照上传题目
- **题目浏览**：按科目、难度、类型浏览题目库
- **题目详情**：查看题目内容、答案、解析
- **答题功能**：学生在线作答并记录
- **搜索功能**：按关键词、标签搜索题目
- **评论系统**：用户可以对题目添加笔记和评论
- **评分系统**：对题目难度和质量进行评分
- **分享功能**：在用户间分享优质题目

### 2. 学习重点管理
- **智能标记**：自动标记难题、易错题
- **优先级推荐**：根据错题率和难度推荐复习内容
- **学习统计**：可视化学习进度和薄弱环节
- **复习提醒**：智能推送需要复习的题目

### 3. 科目板块
- Reading板块
- Writing板块  
- Maths板块
- Thinking Skills板块
- Dashboard（总览仪表板）

### 4. AI功能（预留）
- AI题目解析接口
- AI答题辅导模块
- 智能推荐算法

## 使用场景

学生在日常学习中（纸质书、课堂）遇到难题或易错题时：
1. 使用手机拍照上传题目
2. 系统自动分类和存储
3. 学生可以添加笔记、标记难度
4. 导师可以添加讲解和提示
5. 家长可以查看学习进度
6. 系统智能推荐需要复习的题目

## 技术愿景

- **移动优先**：优化移动端体验
- **响应式设计**：支持多设备访问
- **云端存储**：随时随地访问学习资料
- **可扩展架构**：为AI功能预留接口
- **安全可靠**：保护用户数据和隐私

## 项目约束

- 主要学习活动仍在线下进行（纸质书、课堂）
- 平台聚焦于难题和易错题的管理
- 需要考虑中小学生的使用习惯
- 需要平衡功能丰富度和界面简洁性
